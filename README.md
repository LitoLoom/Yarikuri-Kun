資金移動計算ツール（やりくりくん）
「今月、どの銀行からどの銀行へ、いくら移動させればいいか」をパズル感覚で算出する、シンプルで安全なWebツールです。

🚀 このツールの特徴
あえて銀行連携なし：IDやパスワードを預ける必要がないので、情報漏洩のリスクがありません。
プライバシー重視：入力した残高や支出データは、あなたのブラウザ（LocalStorage）にのみ保存されます。サーバーへ送信されることは一切ありません。
かんたん操作：定例の支出（家賃、クレカ引落など）を一度登録すれば、あとは毎月の残高を入れるだけで「移動が必要な金額」を自動計算します。

🛠 使い方
設定タブ：管理したい銀行名と、毎月の定例支出（どの銀行から引かれるか）を登録します。
計算タブ：現在の各銀行の残高を入力します。
移動指示：残高が不足している口座に対し、メイン口座からいくら移動すべきかが自動的に表示されます。

💡 開発の背景
「家計簿をつけたいわけじゃない、ただ資金ショートを防ぎたいだけなんだ」という自身の悩みを解決するために作成しました。
また、本プロジェクトはスマホのみを使用し、AI（Gemini）との対話を通じて、設計からデプロイまでを完結させた実験的なプロジェクトでもあります。

🛡️ セキュリティとプライバシー
入力データはブラウザの localStorage を使用しています。
アプリを閉じてもデータは保持されますが、他のデバイスやブラウザと共有されることはありません。
作者側であなたの入力内容を把握することは技術的に不可能です。
